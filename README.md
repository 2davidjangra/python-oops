# Python OOP Concepts and Practice Assignments

## Overview

This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts in Python through multiple practice assignments and mini projects.

The project covers:

- Classes and Objects
- Constructors
- Attributes and Methods
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Method Overriding
- Class and Instance Variables

---

## Assignments Included

### 1. Student Management System

Features:
- Store student name, roll number, and marks
- Calculate grade based on marks
- Display student details

Concepts Used:
- Class
- Constructor
- Methods
- Attributes
